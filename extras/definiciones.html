Connecticus - Proyecto Integrador

Etapa 1:

Fecha máxima para tener el proyecto online: Miércoles 13/10/2021.
El proyecto debe ser públicamente accesible desde una página de GitHub Pages personal. Todos los contenidos deberán subirse mediante git. No se podrá utilizar Bootstrap en esta etapa. Deberá utilizarse SASS.
Deberán generar los archivos "alta.html", "contacto.html" y "nosotros.html" y asegurarse de que los vínculos de la barra de navegación permitan acceder a los mismos y al archivo "index.html" ya existente. Todas las páginas deberán tener el mismo encabezado y barra de navegación. Contenido de cada archivo:

index.html:

Es la home que se estuvo trabajando en clase. Deberá contener, como mínimo, la misma estructura HTML y nombres de clases que se definieron en clase. Sobre esa base, deberán adicionarse todos los elementos HTML que se deseen y modificar los estilos CSS existentes (en archivos SCSS), con el fin de personalizar y mejorar el proyecto individual, siempre y cuando el listado siga presentando los productos en formato de tarjetas. Se deberá ajustar el header y barra de navegación, sus colores, etc, en todas las resoluciones y adicionar el logo del proyecto e ícono del carrito. El logo del proyecto deberá estar basado en el nombre Connecticus. Deberá utilizar, como mínimo, los directorios y archivos de SASS ya definidos, si bien se podrán adicionar otros. Todos los estilos utilizados deberán ser escritos en archivos SCSS, los cuales deberán utilizar el único archivo CSS utilizado por el HTML: "main.css".

alta.html:

Deberá contener un formulario con los siguientes campos: nombre, precio, stock, marca, categoria, foto, detalles, envio_sin_cargo. Definir validaciones de HTML 5 y tipos de elementos de manera fundamentada. Todo el diseño de dicha página queda a elección de cada alumno.

contacto.html

Deberá contener un formulario con los siguientes campos: nombre, e-mail y comentarios. Definir validaciones de HTML 5 y tipos de elementos de manera fundamentada. Todo el diseño de dicha página queda a elección de cada alumno.

nosotros.html

Página con contenido y diseño libre. Todo el diseño de dicha página queda a elección de cada alumno.
